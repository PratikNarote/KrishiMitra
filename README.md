# 🌾 KrishiMitra – AI-Based Crop Disease Detection & Weather Advisory

## 📌 Project Overview

KrishiMitra is an AI-powered smart agriculture platform that helps farmers detect crop diseases from leaf images and provides weather-based farming recommendations. The goal is to assist farmers in making better decisions using Artificial Intelligence and modern web technologies.

## 🤖 AI Model

* Model: MobileNetV2 (Transfer Learning)
* Input Size: 224 × 224
* Classes: 15 Crop Diseases
* Loss Function: Categorical Crossentropy
* Optimizer: Adam

---

## 📊 Current Performance

* Training Accuracy: **85.05%**
* Validation Accuracy: **89.54%**
