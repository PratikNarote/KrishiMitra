# 🌱 KrishiMitra

An AI-Based Crop Disease Detection and Weather Advisory System.

## Features

- 🌿 AI Crop Disease Detection
- 🌦️ Weather Advisory
- 🤖 AI Farming Assistant
- 🌾 Crop Recommendation
- 💊 Fertilizer Recommendation
- 📊 Farmer Dashboard
- 🌍 Multilingual Support

## Tech Stack

### Frontend
- React
- Tailwind CSS

### Backend
- Node.js
- Express.js
- MongoDB

### AI
- Python
- TensorFlow
- OpenCV
- FastAPI
